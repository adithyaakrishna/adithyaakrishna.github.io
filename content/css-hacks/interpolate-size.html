interpolate-size

Animate to height: auto

The holy grail of CSS animation. Smoothly transition between a fixed size and an intrinsic keyword like auto. No JavaScript measurement needed.

This content has no fixed height. The browser calculates auto height and smoothly transitions to it. For years this needed JavaScript to measure offsetHeight and animate. With interpolate-size: allow-keywords, it's just CSS. Works with auto, min-content, max-content, and fit-content. Set it on :root to enable everywhere. 
:root {
  interpolate-size: allow-keywords;
}

.content {
  height: 0;
  overflow: hidden;
  transition: height 0.5s ease;
}

.content.open {
  height: auto;
}
