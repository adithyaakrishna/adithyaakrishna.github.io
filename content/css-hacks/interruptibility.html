transition vs keyframes

Make Interactions Interruptible

Transitions interpolate toward the latest state and can be interrupted when you move the cursor away. Keyframe animations run on a fixed timeline and don't retarget once started.

Transition — hover and move away mid-way
Keyframes — runs to completion

Use transitions for interactions that should respond to user intent; use keyframes for staged sequences that run once.

/* Interruptible: reverses when hover ends */
.box {
  transition: transform 0.6s ease;
}
.box:hover {
  transform: translateX(120px);
}

/* Fixed timeline: cannot retarget mid-run */
.box-animated {
  animation: slide 1.2s ease-in-out infinite alternate;
}
